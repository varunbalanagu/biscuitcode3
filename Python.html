PYTHON PROGRAMMING

1. Variables and Data Types

Python is dynamically typed; no need to declare types. Common data types: int, float, str, bool, list, tuple, dict, set. Use type() to check data type. Variables are created when assigned, e.g., x = 10.

2. Operators

Python includes arithmetic (+, -, *, /, **, //, %), comparison (==, !=, >, etc.), logical (and, or, not), assignment (=, +=), and bitwise operators (&, |, ^, ~, <<, >>).

Conditional Statements

if, elif, and else control flow. Indentation matters in Python.
if x > 0:
print("Positive")
elif x == 0:
print("Zero")
else: print("Negative")

Loops

Use for to iterate over sequences and while for condition-based loops. Use break to exit a loop, continue to skip current iteration.

for i in range(5):
print(i)

Functions

Defined using def keyword. Can return values using return. Supports default, keyword, and variable-length arguments.

def greet(name):

return "Hello " + name

Data structures

List: Ordered, mutable, e.g., [1, 2, 3]
Tuple: Ordered, immutable, e.g., (1, 2)
Set: Unordered, unique elements, e.g., {1, 2}
Dict: Key-value pairs, e.g., {"a": 1}


Strings

Use open() with modes: 'r', 'w', 'a', 'b'. Use with statement for automatic file closing. Methods: .read(), .readline(), .write().
with open("file.txt", "r") as f:
data = f.read()

Handled using try-except blocks. Optionally use finally and else. try:
