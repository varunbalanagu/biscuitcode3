CSS Cascading Style Sheets.

CSS Syntax

A CSS rule consists of a selector and a declaration block.
Syntax: selector { property: value; }
Example: h1 { color: red; font-size: 24px; }

Color and Background

color: Sets the text color. Example: color: blue;
background-color: Sets the background color. Example: background-color: yellow;
background-image: Sets an image as the background. Example: background-image: url("img.jpg");
opacity: Sets the transparency level. Example: opacity: 0.5;

Text and Font

font-size: Changes the size of the text. Example: font-size: 16px;
font-family: Sets the font style. Example: font-family: Arial, sans-serif;
font-weight: Defines text thickness. Example: font-weight: bold;
text-align: Aligns text (left, right, center, justify). Example: text-align: center;
text-decoration: Adds effects like underline, overline, etc. Example: text-decoration: underline;
line-height: Sets the space between lines. Example: line-height: 1.5;

Box Model

width: Sets the element's width. Example: width: 300px;
height: Sets the element's height. Example: height: 100px;
padding: Space inside the border. Example: padding: 10px;
margin: Space outside the border. Example: margin: 15px;
border: Sets the border around elements. Example: border: 1px solid black;

positioning

position: Defines how an element is positioned (static, relative, absolute, fixed, sticky).
top, left, right, bottom: Used with positioned elements.

Display and Visibility

display: Specifies how an element is displayed (block, inline, inline-block, none, flex, grid).
visibility: Controls visibility (visible, hidden).
z-index: Controls stacking order.

Flexbox (Layout)

display: flex; enables flex container.
flex-direction: Direction of items (row, column).
justify-content: Aligns items horizontally.
align-items: Aligns items vertically.

Grid (Layout)

display: grid; enables grid layout.
grid-template-columns: Defines column sizes. Example: grid-template-columns: 1fr 2fr;
grid-gap: Space between rows/columns.

Cursor and Pointer

cursor: Changes the cursor when hovering. Example: cursor: pointer;

Transitions and Animations

transition: Smooth change of property values. Example: transition: all 0.3s ease; animation: Defines keyframe-based animations.
