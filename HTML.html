HTML

Basic HTML Structure

HTML documents start with which defines the document type. The "html" tag encloses the entire page content. head contains meta-information like title, charset, and linked resources. body contains the visible content of the web page.

headings

Headings are defined by tags h1 to h6. h1 is the highest level (largest text), and h6 is the lowest. Headings help organize content and improve accessibility and SEO.

Paragraphs and Text Formatting

p defines a paragraph. Text can be emphasized using strong (bold), em (italic), u (underline), and del (strikethrough). These tags help with semantic meaning and styling.

Lists

Ordered lists ol create numbered lists. Unordered lists ul create bulleted lists. List items li are placed inside lists to represent each item.

Links (Hyperlinks)

The a tag creates hyperlinks. The href attribute specifies the destination URL. target="_blank" opens the link in a new tab or window.

Images

The img tag embeds images. src specifies the image URL or path. alt provides alternative text for accessibility. width and height define the image size.

Tables

Tables are defined by table. Rows are tr. Header cells use th, and regular cells use td. Tables organize data in rows and columns.

Forms

form creates a form for user input. action defines where form data is sent. method defines how data is sent (GET or POST). Form elements include inputs, labels, buttons, etc.

Input Types

Different input types: text, password, checkbox, radio, date, email, etc. Each type provides a specific UI and validation behavior. Inputs have name attributes to identify the data submitted.

Div and Span

div is a block-level container used to group content. span is an inline container used for styling parts of text. Neither adds semantic meaning but help with styling and layout.
