C Language

INTRODUCTION

C is considered mother of all programming languages as many later languages have borrowed syntax/features directly or indirectly from the C language like the syntax of Java, PHP, JavaScript, and many other languages that are mainly based on the C language. So, if a person learns C programming first, it will help them to learn any modern programming language as well provide a deeper understanding of the fundamentals of programming and underlying architecture of the operating system like pointers, working with memory locations etc.

Structure of a C Program

A C program typically begins with preprocessor directives that include necessary header files. The execution starts from the main() function, which contains the program’s logic. Inside the main function, variables are declared, and operations or function calls are performed. Every statement ends with a semicolon. The program usually ends with a return statement to indicate successful execution.

2. Data Types

C provides various data types to define variables, such as int for integers, float for floating-point numbers, char for characters, and double for high-precision floating points. These types determine the kind of data a variable can store. Additionally, C supports derived data types like arrays, pointers, structures, and unions. There is also the void type used for functions that do not return any value.

3. Variables and Constants

Variables are used to store data that can change during program execution. They must be declared with a specific data type before use. Constants, on the other hand, are fixed values that cannot be altered once defined. In C, constants are typically declared using the const keyword to prevent modification.

4. Operators

Operators in C are symbols that perform operations on variables and values. These include arithmetic operators for mathematical calculations, relational operators for comparisons, and logical operators for decision-making. There are also assignment, bitwise, and increment/decrement operators. Operators are essential for implementing logic and calculations in a program.

5. Control Statements

Control statements manage the flow of execution in a program. Conditional statements like if, else, and switch allow decision-making based on conditions. Looping statements like for, while, and do-while repeat blocks of code based on conditions. Other control flow statements include break to exit loops and continue to skip iterations.

6. Functions

Functions are self-contained blocks of code that perform specific tasks. They help in organizing code, improving readability, and promoting reuse. Functions can accept inputs called parameters and may return a value. C provides both built-in library functions and allows the creation of user-defined functions.

7. Arrays

Arrays are collections of elements of the same data type stored in contiguous memory locations. They allow multiple values to be stored and accessed using a single variable name with an index. Arrays can be one-dimensional or multi-dimensional, like matrices.

8. Strings

In C, strings are represented as arrays of characters ending with a null character (\0). They are used to store and manipulate sequences of characters. Standard string-handling functions are available in the C library to perform operations like copying, comparing, and concatenating strings.

9. Pointers

Pointers are variables that store the memory addresses of other variables. They enable dynamic memory management and are crucial for advanced features like passing parameters by reference. Pointers are also used for working with arrays and functions efficiently.

10. Structures

Structures allow grouping variables of different data types under one name. They are useful for creating complex data models, such as records or objects. Each variable inside a structure is called a member, and structures are widely used in file handling and system programming.

11. File Handling

File handling in C allows programs to read from and write to files stored on disk. It involves using file pointers and functions to open, read, write, and close files. This capability is essential for tasks like data logging, configuration management, and persistent storage.

DATA TYPES

char Size: 1 byte Description: Used to store a single character (e.g., 'A', 'z').

int Size: 4 bytes (typical) Description: Used to store whole numbers, both positive and negative.

short Size: 2 bytes Description: Used to store smaller-range integers.

long Size: 4 or 8 bytes (depends on system) Description: Used to store large integers.

long long Size: 8 bytes Description: Used to store very large integers.

float Size: 4 bytes Description: Used to store decimal numbers with single precision (around 6–7 digits).

double Size: 8 bytes Description: Used to store decimal numbers with double precision (around 15–16 digits).

long double Size: 10 to 16 bytes (depending on compiler) Description: Used to store very large or very precise decimal numbers.

void Size: 0 bytes Description: Represents no data type; often used for functions that return nothing.

_Bool / bool (available in C99 and later) Size: 1 byte Description: Used to store boolean values (true or false).
