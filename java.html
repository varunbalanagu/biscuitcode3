JAVA language

Variables and Data Types

Java is statically typed; every variable must have a declared type. Primitive types: int, float, double, char, boolean, byte, short, long. Non-primitive types include String, Arrays, Objects. Example: int x = 10; String name = "Java";

Operators

Includes arithmetic (+, -, *, /, %), comparison (==, !=, <, >), logical (&&, ||, !), assignment (=, +=, etc.), bitwise (&, |, ^, ~, <<, >>). Used to perform operations on variables and values.

Conditional Statements

Java uses if, else if, else, and switch for decision-making. Syntax includes braces {} even for single-line blocks.

if (x > 0) {
System.out.println("Positive");
} else {
System.out.println("Not Positive"); }

Loops

Java supports for, while, and do-while loops. Use break to exit a loop and continue to skip the current iteration. Enhanced for-each loop is used for arrays and collections.

Functions (Methods)

Declared inside classes using returnType methodName(params). Use static for class methods. Can be overloaded with different parameter lists.
int add(int a, int b) {
return a + b; }

Arrays and Collections

Arrays: Fixed size, e.g., int[] arr = new int[5]; Collections: Resizable structures like ArrayList, HashMap, HashSet. Use java.util.* for collection classes.

Strings

Immutable and handled by the String class. Common methods: .length(), .charAt(), .substring(), .equals(), .compareTo(). Use StringBuilder or StringBuffer for mutable strings.

File Handling

Use classes from java.io or java.nio.file. Common classes: File, FileReader, FileWriter, BufferedReader. File myFile = new File("data.txt");
if (myFile.exists()) {
... }

OOP (Object-Oriented Programming)

Core principles: Encapsulation, Inheritance, Polymorphism, Abstraction. Use class to define objects, extends for inheritance. Access modifiers: private, public, protected, default.

Inheritance

class Animal {
  void sound() {
    System.out.println("Animal makes sound");
  }
}

class Dog extends Animal {
  void bark() {
    System.out.println("Dog barks");
  }
}
